Upgrade Instructions for local installations

Version 2.0.20

The JanBan app can now be run from the internet instead of a local installation.

Your data will not be exposed, because the app still keeps everything within the context of Outlook.

This solves the issue for people who cannot - or do not want to - install software locally.

Changing to this new setup is just a matter of using another url as the home page of your Outlook Home Folder:

  • Right-click your Outlook Home Folder and then click Properties. Select the Home Page tab in the Properties dialog box.
  • In the Address box, type: https://janware.nl/janban. Click to select the Show home page by default for this folder check box and then click OK.
  • Or download and run the install.cmd batch file, which does this for you.

    • But of course you can stick to your local version. That's up to you.
